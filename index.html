¿Que es programación?

1La programación de software

Se trata de un conjunto de herramientas y utilidades que permiten a los programadores desarrollar programas informáticos utilizando diferentes lenguajes de programación o bases de datos. Incluye de manera general: editores de texto, compiladores, intérpretes, enlazadores y depuradores. Es decir el arte del proceso por el cual se limpia, codifica, traza y protege el código fuente de programas computacionales, en otras palabras, es indicarle a la computadora lo que tiene que hacer.

Lenguaje Java

1Que es Java:

Java es un lenguaje de programación y una plataforma informática que fue comercializada por primera vez en 1995 por Sun Microsystems. Hay muchas aplicaciones y sitios web que no funcionarán, probablemente, a menos que tengan Java instalado y cada día se crean más. Java es rápido, seguro y fiable. Desde ordenadores portátiles hasta centros de datos, desde consolas para juegos hasta computadoras avanzadas, desde teléfonos móviles hasta Internet, Java está en todas partes, si es ejecutado en una plataforma no tiene que ser recompilado para correr en otra. Java es, a partir de 2012, uno de los lenguajes de programación más populares en uso, particularmente para aplicaciones de cliente-servidor de web, con unos diez millones de usuarios reportados.2​3​ El lenguaje de programación Java fue desarrollado originalmente por James Gosling, de Sun Microsystems (constituida en 1983 y posteriormente adquirida el 27 de enero de 2010 por la compañía Oracle),4​ y publicado en 1995 como un componente fundamental de la plataforma Java de Sun Microsystems.

Tipos de datos Java

1Los Tipos De Datos Java:

En programación, un tipo de dato informático o simplemente tipo es un atributo de los datos que indica al ordenador (y/o al programador) sobre la clase de datos que se va a trabajar. Esto incluye imponer restricciones en los datos, como qué valores pueden tomar y qué operaciones se pueden realizar.

Operadores de Java

1Operadores:

Los operadores son símbolos que indican cómo se deben manipular los operandos. ... Los operandos pueden ser constantes, variables o llamadas a funciones, siempre que éstas devuelvan algún valor.

Ejemplo de Programacion en java

1Credito:

El valor de la cuota depende del tiempo que tardará el deudor en pagar toda la deuda y de la tasa de interés. Por ejemplo, para una deuda de $1.200.000, por un año, con una tasa del 10%, se deben pagar doce cuotas mensuales de $220.000, que corresponden a $100.000 por pago de capital y $120.000 por pago de interés. Al final del año el deudor paga $2.640.000, $1.200.000 correspondientes a pago de capital y $1.440.000 por los intereses (un negocio muy lucrativo, 120%). Se requiere calcular el valor de la cuota de un crédito conociendo el monto del crédito, la tasa de interés y la cantidad de meses de la deuda. También se requiere saber cuánto dinero corresponde a pago de capital y cuánto a pago de interés, para cada cuota y al final del crédito. Además se requiere conocer el porcentaje total de ganancias del crédito. Desprecie los centavos al realizar los calculos de porcentajes y considere sumas grandes de dinero.
